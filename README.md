Lesson App Frontend
Description: Vue.js interface to browse lessons, add them to a cart, and checkout. 
Tech : Vue.js, HTML, CSS, JavScript, Fetch API
GitHub Repo: https://github.com/Zoz0chan/lesson-app-frontend
Live App(GitHub Pages): https://zoz0chan.github.io/lesson-app-frontend/
Backend connection : Fetches data from https://lesson-app-backend-pbex.onrender.com

NOTE: The frontend works fully when the backend API is live
All requests are handled via Fetch API to the backend
